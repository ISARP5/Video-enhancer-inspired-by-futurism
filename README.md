## 📡 SYSTEM OVERVIEW
Una extensión inyectora de aceleración visual diseñada sin plástico genérico y estructurada en Y2K Webcore y Glassmorphism.

Esta extensión monitoriza activamente cualquier `<video>` incrustado en el navegador utilizando `IntersectionObserver` y acelera a través de shaders de filtrado a su GPU usando la variable directriz `will-change: filter`. Utiliza un filtro inteligente `Contrast Adaptive Sharpening (CAS)` calculado puramente a base de deformación en matrices SVG.

## 💽 NETWORK FEATURES 
- **Vanilla / Golden Auto / Custom:** Disparadores interactivos del motor Visual. Golden Auto recalibra instantáneamente la visualización en los porcentajes perfectos definidos tras cálculos intensivos (15% Sharpening, 108% Contraste, 112% Saturación).
- **Single Source of Truth SSoT:** El motor corre independientemente por cada ventana detectando transiciones y conectándose persistentemente con `chrome.storage.local`.
- **Pixel-Perfect Aesthetic:** Una interfaz completamente dibujada con `backdrop-filter: blur`, paleta arquitectónica pálida, tipografía modular (Helmet, Space Mono) e inyecciones cinéticas realistas de peso en cada botón presionado.

## 🔌 INSTALACIÓN (MODO DESARROLLADOR)

1. Descarga el repositorio completo en formato `ZIP` y descomprímelo o haz un Clon Vía Git.
2. Abre tu navegador basado en Chromium (Google Chrome, Brave, Arc, Edge, Opera, etc).
3. Navega al apartado de extensiones e **inicia la url**: `chrome://extensions/`
4. Activa el marco `[Modo Desarrollador]` o `[Developer mode]` en la cabecera del panel derecho.
5. Inyecta los archivos haciendo click en **"Cargar extensión sin empaquetar"** (Load Unpacked) y apunta visualmente a la carpeta descomprimida. 
6. Listo. Accede en el anclaje de tu barra de marcadores.

Esta extensión adapta el concepto matemático de convolución a la API web estándar de filtros SVG (`feConvolveMatrix`).
